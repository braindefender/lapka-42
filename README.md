# Лапка 42

Беспроводная эргономичная сплит-клавитура для тех, у кого лапки (:

**Информация:**

|                         |                                 |
| :---------------------- | :------------------------------ |
| **Количество клавиш**   | 42                              |
| **Тип матрицы**         | Диодная                         |
| **Направление диодов**  | Col2Row                         |
| **Совместимые MCU**     | nRF52840 (SuperMini, nice!nano) |
| **Совместимые АКБ**     | 401230                          |
| **Совместимые свитчи**  | Choc v1/v2                      |
| **Свитч-спейсинг**      | Choc (18×17mm)                  |
| **Расположение клавиш** | Splay                           |

**Структура репозитория:**

| Директория | Описание                                              |
| ---------: | :---------------------------------------------------- |
|        3MF | Проект OrcaSlicer (для примера параметров печати)     |
|     Assets | Изображения и фото                                    |
|      Build | Описание [процесса сборки](./Build/BUILD_GUIDE_RU.md) |
|     Fusion | Проект Autodesk Fusion 360                            |
|    Gerbers | Gerber-файлы для производства плат                    |
|      KiCad | Проект KiCad (схема, плата, компоненты)               |
|        STL | 3D-модели частей корпуса                              |

**Части корпуса:**
|   Файл | Описание                                                      |
| -----: | :------------------------------------------------------------ |
| Bottom | Нижняя часть корпуса, имеет вырезы под hotswap-сокеты и диоды |
|  Cover | Крышка для АКБ и кнопки Reset                                 |
|  Frame | Верхняя часть корпуса, для тайп-кластера                      |
|  Thumb | Верхняя часть корпуса, для тамб-кластера                      |

**Разметка пинов:**

> C — колонки, R — строки, нумерация слева направо, сверху вниз

|  Пин  | Левая | Правая |
| :---: | :---: | :----: |
|  C0   |  D4   |  D10   |
|  C1   |  D5   |  D16   |
|  C2   |  D6   |  D14   |
|  C3   |  D7   |  D15   |
|  C4   |  D8   |  D18   |
|  C5   |  D9   |  D19   |
|       |       |
|  R0   |  D2   |  D21   |
|  R1   |  D3   |  D20   |
|  R2   |  D16  |   D8   |
|  R3   |  D10  |   D9   |
